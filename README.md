# 🛡️ FocusGuard — App Usage Controller

Apne phone ke apps ka use control karein!

---

## ✅ Features

- 📱 Installed apps ki list dekhein aur select karein
- ⏱️ Har app ke liye daily time limit set karein (15 min se 4 ghante tak)
- 🔔 Har 10 minute mein reminder notification
- 🔒 Limit khatam hone par automatic blocking screen
- 🔓 Emergency unlock (+15 min) agar zaroorat ho
- 📊 Aaj ka usage live dekhein
- 🔄 Phone reboot ke baad bhi kaam karta hai

---

## 📥 APK Download Karne ke Steps (FREE)

### Step 1: GitHub Account Banayein (Ek baar)
1. **github.com** website kholein
2. "Sign Up" par click karein
3. Email, username, password dalein — account ban jayega
4. Email verify karein

### Step 2: Is Project Ko Copy Karein
1. GitHub login karein
2. Upar "+" icon → "New repository" click karein
3. Repository name: `focusguard` likhein
4. "Public" select karein
5. **"Create repository"** click karein

### Step 3: Saari Files Upload Karein
1. Naye repository mein **"uploading an existing file"** link click karein
2. **ZIP file extract karein** apne computer pe
3. Extracted folder ke andar se **sabhi files aur folders** select karein
4. GitHub pe drag & drop karein
5. Scroll karein neeche → **"Commit changes"** click karein

### Step 4: APK Automatically Build Hoga
1. Repository mein **"Actions"** tab click karein
2. Aapko "Build FocusGuard APK" workflow dikhega
3. **Yellow circle** = Build chal raha hai (5-10 min wait karein)
4. **Green checkmark** ✅ = APK ready hai!
5. Workflow par click karein → Neeche "Artifacts" mein **"FocusGuard-APK"** download karein

## 📱 Phone Pe Install Karne ke Steps

### Pehle "Unknown Sources" allow karein:
1. Phone Settings kholein
2. **Security** ya **Privacy** mein jayein
3. **"Install unknown apps"** ya **"Unknown sources"** ON karein
4. Ya: File Manager app ke liye allow karein

### APK Install Karein:
1. Downloaded ZIP file kholein, andar `app-debug.apk` milega
2. APK file par tap karein
3. **"Install"** par tap karein
4. Done! ✅

## 🚀 FocusGuard Use Karne ke Steps

### Pehli baar setup:
1. FocusGuard app kholein
2. **"Setup & Permissions"** button dabayein
3. Teen permissions dene honge:

   **Permission 1: Usage Access**
   - Settings mein FocusGuard dhundein
   - Toggle ON karein → Wapas aayein

   **Permission 2: Overlay (Display over apps)**
   - FocusGuard ke liye toggle ON karein → Wapas aayein

   **Permission 3: Battery Optimization**
   - "Allow" dabayein taaki background mein chale

4. Dobara "Setup" dabayein — "✅ Sab Ready Hai!" dikhega

### Apps ko monitor karna:
1. Main screen pe apni apps ki list dikhegi
2. Jis app ko control karna ho uske saamne **Toggle ON** karein
3. App ke naam par **tap** karein → settings kholein
4. **Time limit** set karein (slider ya quick buttons se)
5. **"Settings Save Karein"** dabayein

### Kya hoga:
- ✅ Har 10 minute pe notification aayega
- 🔒 Limit khatam hone par blue blocking screen aayega
- 🏠 "Home Pe Jayein" se phone ke home screen pe jayenge
- 🔓 Emergency mein "+15 min" unlock kar sakte hain

## ❓ Agar APK Build Fail Ho (Red X)

1. Actions tab mein failed build par click karein
2. Error message dekhein
3. Koi bhi error copy karein aur Claude ko bhejein — main fix kar dunga!
